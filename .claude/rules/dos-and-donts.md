# Do's and Don'ts

## Do

- Make the hero section immediately impressive (gradient text, animations, particle effects)
- Add smooth scroll behavior for anchor navigation
- Use `loading="lazy"` on images below the fold
- Include a favicon and proper page title
- Test all interactive elements for keyboard accessibility
- Use CSS Grid and Flexbox for layouts
- Add subtle hover states to everything clickable
- Keep animations at 60fps

## Don't

- Use CSS frameworks (no Tailwind, Bootstrap, etc.)
- Use jQuery or any JS frameworks/libraries
- Use inline styles — all styling in CSS files
- Hardcode colors or spacing — use CSS custom properties
- Remove existing comments or documentation
- Add npm/node dependencies — zero-dependency static site
- Use placeholder images — generate real ones or use CSS art
- Ship sections that look flat or generic
- Forget focus styles for keyboard navigation
