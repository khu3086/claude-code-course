# Accessibility

- **Contrast**: minimum 4.5:1 for body text, 3:1 for large text
- All images have descriptive `alt` text
- Interactive elements are focusable and have visible focus indicators
- Skip-to-content link as the first focusable element
- ARIA attributes on custom interactive widgets
- All interactive elements must work via keyboard
- Respect `prefers-reduced-motion` — disable non-essential animations
