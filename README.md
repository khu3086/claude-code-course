# Claude Code Course

> ⚠️ **This repository is for practice purposes only.**

This repo contains projects built while learning and experimenting with AI-assisted coding using Claude.

## Projects

### 🏔️ Mountain Explorer
An interactive scrolling website that lets you explore mountain flora and fauna at every altitude — from the summit (8,849m) down to the foothills. Inspired by [neal.fun/deep-sea](https://neal.fun/deep-sea/).

**To run locally:**
```bash
npx serve mountain-explorer
```
